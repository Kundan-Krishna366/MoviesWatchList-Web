# Movie App

A React-based movie application using the TMDB API to browse and search movies.  
Users can add favourites and watch movies via VidSrc.

## Live Demo
https://watchonlymovies.netlify.app/

## Note
This application currently provides the best experience on desktop and laptop browsers.

## Features
- Popular & search movies
- Add/remove favourites
- Watch movies (VidSrc embed)
- Fast & responsive UI
